Module Contents

.. py:class:: Timer(name: str = 'Timer', unit: str = 'ms')

   
   Timer for measuring code execution time with statistics.

   Uses MATLAB-style tic/toc naming.

   .. rubric:: Example

   ..

      ..

         ..

            timer = Timer("inference")
            for _ in range(100):
            ...     timer.tic()
            ...     model.forward(x)
            ...     timer.toc()
            timer.print_stats()
            [inference:] mean=1.234ms, std=0.123ms, ..., total=123.400ms (n=100)


   Or use as context manager:
   ==========================

   ..

      ..

         ..

            with timer:
            ...     model.forward(x)


   .. py:attribute:: name
      :value: 'Timer'



   .. py:attribute:: unit

   .. py:method:: tic() -> None

      
      Start the timer (MATLAB-style).


   .. py:method:: toc() -> float

      
      Stop the timer and record the elapsed time (MATLAB-style).

      :returns: The elapsed time in seconds.

      :raises RuntimeError: If the timer was not started.


   .. py:attribute:: start


   .. py:attribute:: stop

      Reset all recorded times.
